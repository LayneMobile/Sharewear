# Sharewear
Utility Library for Android Wear

## Download
```groovy
dependencies {
    compile 'com.laynemobile.sharewear:sharewear:1.0.1'
}
```

## Build

To build:

```bash
$ git clone git@github.com:LayneMobile/Sharewear.git
$ cd Sharewear/
$ ./gradlew build
```
