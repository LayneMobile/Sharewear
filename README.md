# Sharewear
Utility Library for Android Wear

## Download
```groovy
repositories {
    maven { url 'http://dl.bintray.com/laynemobile/android' }
}

dependencies {
    compile 'com.laynemobile.sharewear:sharewear:1.0.0'
}
```

## Build

To build:

```bash
$ git clone git@github.com:LayneMobile/Sharewear.git
$ cd Sharewear/
$ ./gradlew build
```
